## Instal from source code

```bash
git clone https://github.com/WagnoLeaoSergio/portfolio_auto_analysis
cd portfolio_auto_analysis
pip3 install -r requirements.txt
```

## Usage

```bash
cd portfolio_auto_analysis
streamlit run dashboard.py
```
